# Reia - Python 3.14 버전

Rust로 작성된 Reia 게임 엔진 확장을 Python 3.14로 변환한 버전입니다.

자세한 문서는 [docs](./docs/) 폴더를 참조하세요.

## 빠른 시작

```bash
# 의존성 설치
pip install -r requirements.txt

# 개발 모드로 설치
pip install -e ".[dev]"
```
